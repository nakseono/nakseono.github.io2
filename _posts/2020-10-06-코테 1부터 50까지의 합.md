정말 가고 싶었던 기업에서 코딩테스트를 망치고 (ㅠㅠㅠㅠㅠㅠ..)
이후 혼자 공부할 때 기억을 더듬어 못풀었던 문제를 다시 풀어보려고 한다.

## 문제 설명
1부터 50까지의 수를 더하는 함수를 작성하라.  

## 제한사항
- 10의 배수(10, 20, 30, ...)인 상황에서는  
"10까지의 합 55", "20까지의 합 *", ...
이런식으로 콘솔에 찍히는 함수를 작성하라.

---
# 문제풀이
## 의사 코드 (Pseudo-Code)
1. for문을 통해 1부터 50까지 += i 를 해주면 되겠다.
2. 이 때, i 가 10의 배수이면. 즉 i % 10 === 0 이라면 console.log를 찍어주자.

### 정리된 코드
```js
function solution() {
  let sum = 0;
  for (let i = 1; i <= 50; i++) {
    sum += i;
    if (i !== 0 && i % 10 === 0) {
      console.log(`${i}까지의 합 ${sum}`);
    }
  }
}
```
